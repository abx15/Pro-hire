## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in ProHire, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please send an email to [INSERT CONTACT EMAIL] with the subject line "Security Vulnerability".

In your report, please include:

- A detailed description of the vulnerability.
- Steps to reproduce the vulnerability.
- Any relevant logs or error messages.

We will acknowledge receipt of your report within 48 hours and will work to verify and resolve the issue as quickly as possible.

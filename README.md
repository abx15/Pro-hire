# Hiremate

This is the Hiremate frontend built with Vite, React, TypeScript and TailwindCSS.

## Getting started

Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
